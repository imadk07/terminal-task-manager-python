# Terminal Task Manager (Python)

A terminal-based task manager built with Python using object-oriented programming principles.

## Features

- Add tasks
- View tasks
- Edit tasks
- Remove tasks
- Input validation
- Menu-driven CLI interface
- Save tasks to JSON
- Load tasks from JSON

## Concepts Used

- Classes & Objects
- Instance Methods
- Class Methods
- Static Methods
- CRUD Operations
- Validation
- Loops & Conditionals
- String Representations (`__str__`, `__repr__`)
- JSON Serialization & Persistence
- File Handling
- `__dict__`
- `**` Unpacking

## Technologies

- Python 3

## How to Run

```bash
python3 tasks.py
```
